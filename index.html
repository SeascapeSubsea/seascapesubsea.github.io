Json File

The JSON configuration file used in BlueOS is responsible for defining the communication layout and routing for MAVLink-enabled devices within the system. It outlines how devices like flight controllers are connected, specifying details such as baud rates, endpoint addresses, and routing paths. This file is primarily used by the BlueOS MAVLink Manager to establish and manage connections between onboard components and external interfaces like QGroundControl or MAVProxy, ensuring that data flows correctly between subsystems.
